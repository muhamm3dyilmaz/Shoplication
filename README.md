KULLANIMI:
Projeyi bilgisayarınıza indirdikten sonra rardan çıkarın.
Projeyi Visual Studio Code ile açın.
VS Code içerisinde terminal açın.
Terminale 'cd .\ShopApp.webui\' yazın ve enter'i tuşlayın.
Ardından teminale 'dotnet watch run' yazın ve enter'i tuşlayın.
